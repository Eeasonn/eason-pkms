- **产品思维与方案设计**
  - 学习如何从会搭 AI 工具过渡到会想、会判断、会打磨一个有 sense 的 AI 应用，掌握产品思维的核心理念和实践方法。
  - 原文: `raw/courses/easy-vibe/stage-1/appendix-a-product-thinking/index.md`
  - 在线: https://datawhalechina.github.io/easy-vibe/zh-cn/stage-1/appendix-a-product-thinking/

### [[appendix-b-common-errors]]

  - 学习如何高效地向 AI 提问来解决开发中的各种报错问题，掌握截图、描述、定位问题的标准流程，让 AI 成为你的调试助手。
  - 原文: `raw/courses/easy-vibe/stage-1/appendix-b-common-errors/index.md`
  - 在线: https://datawhalechina.github.io/easy-vibe/zh-cn/stage-1/appendix-b-common-errors/

### [[appendix-consumer-scenarios]]

  - 原文: `raw/courses/easy-vibe/stage-1/appendix-idea-sources/index.md`

### [[appendix-jobs-to-be-done]]
- **用 Jobs to Be Done 找到用户真正想完成的事**
  - 面向零基础读者的 Jobs to Be Done 入门文章。理解用户不是在买功能，而是在特定场景里“雇用”你的产品完成进展，并学会用 JTBD 拆解产品方向、访谈问题与 AI 提示词。
  - 原文: `raw/courses/easy-vibe/stage-1/appendix-jobs-to-be-done/index.md`
  - 在线: https://datawhalechina.github.io/easy-vibe/zh-cn/stage-1/appendix-jobs-to-be-done/

### [[appendix-mom-test]]
- **The Mom Test：如何通过用户访谈验证需求**
  - 面向零基础读者的 The Mom Test 入门文章。学会避免礼貌性反馈，围绕真实行为、具体事实和已有问题做用户访谈，把“听起来不错”变成更可靠的需求判断。
  - 原文: `raw/courses/easy-vibe/stage-1/appendix-mom-test/index.md`
  - 在线: https://datawhalechina.github.io/easy-vibe/zh-cn/stage-1/appendix-mom-test/

  - 原文: `raw/courses/easy-vibe/stage-1/complete-project-practice/index.md`
  - 在线: https://datawhalechina.github.io/easy-vibe/zh-cn/stage-1/complete-project-practice/

### [[finding-great-idea]]
