## 核心要点

- 完整项目阶段：原型 → MVP → Beta → 正式版
- 每个阶段对应不同用户量和稳定性要求
- AI 项目特有的部署问题：API 成本、限流、延迟
- 上线前必须有降级方案（AI 挂了怎么办）

## 关键洞察

> AI 项目的工程难点不是 AI，而是'AI 不可靠'——必须有降级、降本、降速的工程化方案。

## 可立刻应用的行动

- [ ] 给你的项目画一个降级方案树
- [ ] 估算每月 API 成本并设置预算告警
