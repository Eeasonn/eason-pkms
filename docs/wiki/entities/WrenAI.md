# WrenAI

## 概述

WrenAI 是开源 **ChatBI** 工具，代表语义层的"消费端"：
- 自然语言提问 → 查语义模型 → 生成 SQL → 返回结果

## 定位

**语义层的消费者**，不是生产者：
- 能定义语义模型（MDL）
- 能自然语言问数
- 不能展示语义层怎么建（AI 识别、置信度、人工审核）

## 在 Demo 中的作用

展示"语义层建好后，业务人员可以这样用"——自然语言问数、指标浏览。
