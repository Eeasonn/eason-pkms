# 数据技能体系（七层结构）

## 总览

本页是 Eason 数据技能知识库的总览导航，涵盖从基础技能到领域实践的完整数据价值链。

## 七层结构

### 00-基础技能层
数据从业者的通用底层能力，包括 SQL 查询、Python 数据处理、AI 素养以及数学与统计学基础。这一层是向上攀登的根基。

### 01-数据认知层
建立对数据的系统认知，涵盖数据类型与结构、数据建模方法、数据生命周期、数据质量管理以及主数据与元数据管理。

### 02-数据工程层
数据价值链的基建层，聚焦数据采集、存储系统、数据处理、数据治理、数据架构、数据可观测性等核心工程主题。

### 03-数据分析层
将数据转化为洞察的核心方法论层，包括指标体系、描述/诊断/预测/分析方法、统计学基础、实验设计与因果推断。

### 04-数据应用层
数据价值的直接变现层，涵盖数据产品设计、数据可视化、数据驱动决策、AB 测试、增长黑客等内容。

### 05-数据战略层
从组织视角看待数据，包含数据战略规划、数据组织架构、数据治理体系、数据文化与伦理合规。

### 06-领域实践层
垂直行业和业务场景中的数据实践，覆盖互联网、金融、零售电商、SaaS、智能制造等领域。

## 学习路径建议

### 入门路线
1. 先建立整体框架：数据认知
2. 夯实工具技能：基础技能（重点 SQL 与 Python）
3. 理解数据如何流动：数据工程
4. 尝试第一次数据分析：数据分析

### 进阶路线
1. 深耕工程能力：数据工程（深入数据治理、Lakehouse）
2. 精通分析方法论：数据分析（统计推断、实验设计）
3. 掌握数据产品化思维：数据应用
4. 拓展战略视野：数据战略

### 实战路线
1. 选定目标行业：领域实践
2. 结合业务场景做完整项目
3. 用数据应用方法验证效果（AB 测试、指标体系监控）
4. 复盘沉淀为个人案例库
